"next" 链接类型

next 关键字可用于 link, a 和 area 元素。该关键字会创建 超链接。

next 关键字表示该文档是一个文档序列的一部分， 而且该链接指向逻辑上位于当前文档之后的那个文档。

当 next 关键字用于 link 元素时， 用户代理应该实现一个 Resource Hints 中描述的处理模型， 也就是说这样的链接应该被当做使用了 dns-prefetch, preconnect, prefetch 或 prerender 关键字来处理。用户代理希望使用哪个资源暗示是实现相关的； 例如用户代理为了节省数据、电池或计算量可能会希望使用代价较小的 preconnect 暗示， 或者希望基于对类似场景中以前的用户行为的启发式分析来选取一个资源暗示。
