Transfer ( input, targetRealm )

  1. 如果 input 有一个 [[ArrayBufferData]] 内部槽，那么：

    1. 令 output 为 targetRealm 中一个新的 ArrayBuffer 对象， 其 [[ArrayBufferByteLength]] 内部槽的值为 input 的 [[ArrayBufferByteLength]] 内部槽的值， 其 [[ArrayBufferData]] 内部槽的值为input 的 [[ArrayBufferData]] 内部槽的值。

    2. 执行 ! DetachArrayBuffer(input)。

    3. 返回 output。

  2. 返回 ? input.[[Transfer]](targetRealm)。
