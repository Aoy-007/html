Blob 和 FileList 对象的猴子补丁

本猴子补丁将在适当的时候删除。见 w3c/FileAPI issue 32.

Blob 对象是 可克隆对象。

每个 Blob 对象的 [[Clone]] 内部方法，给定 targetRealm 且忽略 memory 时，必须执行这些步骤：

  1. 如果 this 处于 closed 状态，则抛出一个 "DataCloneError" DOMException。

  2. 返回 targetRealm 中 this 的一个新实例。对应于相同的底层数据。

FileList 对象是 可克隆对象。

每个 FileList 对象的 [[Clone]] 内部方法，给定 targetRealm 和 memory时，必须执行这些步骤：

  1. 令 output 为 targetRealm 中一个新的 FileList 对象。

  2. 对于 this 中的每一个 file，在 output 的 File 列表的尾部添加 ? StructuredClone(file, targetRealm, memory)。

  3. 返回 output。
