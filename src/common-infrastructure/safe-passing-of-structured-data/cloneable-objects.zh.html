可克隆的对象

可克隆的对象 支持在 事件循环之间克隆。即它们支持在文档和 worker 的边界进行克隆， 包括不同 域 的文档之间。并非所有对象都是 可克隆的对象，也并不是 可克隆的对象 的所有方面都必须在克隆时保留。

平台对象 有如下的内部方法：

[[Clone]] ( targetRealm, memory )

除非另有说明，调用 [[Clone]] 内部方法必须抛出一个 "DataCloneError" DOMException。（ 默认情况下 平台对象 不是 可克隆对象。）

属于 可克隆对象 的 平台对象 有一个 [[Clone]] 内部方法，该方法被指定运行一系列的步骤。这些步骤的结果必须是 抛出异常或者在targetRealm中创建一份 this 的克隆。 具体如何克隆取决于这些对象如何去定义它。

JavaScript 标准中定义的对象直接交由 StructuredClone 抽象操作。
