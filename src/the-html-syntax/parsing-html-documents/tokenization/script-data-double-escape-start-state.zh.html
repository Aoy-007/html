脚本数据双重转义开始状态

使用掉 下一个输入字符：

U+0009 CHARACTER TABULATION (tab)
U+000A LINE FEED (LF)
U+000C FORM FEED (FF)
U+0020 SPACE
U+002F SOLIDUS (/)
U+003E GREATER-THAN SIGN (>)
如果 temporary buffer 是字符串 "script", 则切换到 脚本数据双重转义状态。 否则, 切换到 脚本数据转义状态。 把 当前输入字符 作为字符标记发出。
ASCII 大写字母
把 当前输入字符 （添加一个 0x0020 到该字符的代码点） 的小写版本追加到 temporary buffer。 把 当前输入字符 作为字符标记发出。
ASCII 小写字母
把 当前输入字符 追加到 temporary buffer。 把 当前输入字符 作为字符标记发出。
任何其他情况
在 脚本数据转义状态 重新消耗 该字符。
