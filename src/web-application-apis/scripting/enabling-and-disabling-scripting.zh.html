启用和禁用脚本

在 浏览环境 中下列条件都为真时， 脚本处于启用状态：

  • 用户代理支持脚本。
  • 用户当前没有为这个 浏览环境 禁用脚本。 （用户代理可能为用户提供全局或更细粒度的禁用脚本的选项，例如根据每个域来禁用）
  • 浏览环境 的 活动文档 的 激活沙盒标识 没有设置它的 沙盒脚本浏览上下文标志。

在 浏览环境 中上述任何条件为假时， 脚本处于禁用状态 （例如当脚本没有 启用 时）。

如果一个节点 node 的 节点文档 有一个 浏览上下文， 且该 浏览环境 中 启用了脚本，那么 该节点 node 就 启用了脚本。

如果不存在这样的 浏览环境， 或者该 浏览环境 中 脚本被禁用，那么该节点就 禁用了脚本.
