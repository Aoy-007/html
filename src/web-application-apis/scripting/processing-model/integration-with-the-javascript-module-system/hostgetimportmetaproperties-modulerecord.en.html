HostGetImportMetaProperties(moduleRecord)

The import.meta proposal contains an implementation-defined HostGetImportMetaProperties abstract operation. User agents must use the following implementation:

  1. Let module script be moduleRecord.[[HostDefined]].

  2. Let urlString be module script's base URL, serialized.

  3. Return « Record { [[Key]]: "url", [[Value]]: urlString } ».
