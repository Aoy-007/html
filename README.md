## 关于本项目

[本项目][whatwg-cn/html] 的初衷是尽可能维护最新的内容翻译，而不是基于某个固定版本进行翻译。
即保持 Live Standard 的活力，但由于翻译时间不同**会出现一致性问题请读者注意**。

由于 [whatwg/html][whatwg/html] 标准的更新频率非常快，翻译内容也需要经常更新。
因此 **欢迎通过 Pull Request 参与翻译，也欢迎通过 Issue 指出翻译问题**。
